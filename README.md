# subline-site

[Subline](https://github.com/jsonpassion/BestPlayer)(서브라인) — macOS용 온디바이스 자막 플레이어 —
의 소개 웹사이트와 법적 문서 저장소.

배포: GitHub Pages (`main` 브랜치의 `/docs`)
→ https://jsonpassion.github.io/subline-site/

## 구조

```
docs/
├── index.html    # 랜딩 페이지 (기능, 단축키, 데모, FAQ)
├── privacy.html  # 개인정보 처리방침 (한국어/영어)
├── terms.html    # 이용약관 (한국어/영어)
└── styles.css    # 다크 시네마 테마 공용 스타일
```
